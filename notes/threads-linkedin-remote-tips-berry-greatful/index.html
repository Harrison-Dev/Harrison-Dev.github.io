TL;DR

  • 只要英語能工作溝通，LinkedIn 的遠距職缺是「寶庫」。
  • 三招提升成功率：
    1) 搶佔先機（黃金 30 人法則：申請人數 < 30 時盡快投遞）；
    2) 英語程度選「Conversational」即可（多數職缺不必 Native）；
    3) 善用 Quick Apply + 開啟職缺通知。
  • Web3 多半時區友善；非 Web3 常見要求配合當地時區或設置「座標美國」。
  • 謹慎防詐與無薪實習，投遞前查證公司、合約、薪資與地點/簽證條款。

—

重點摘要
1) 貼文主旨

  • 英語溝通沒問題者，可大量挖掘 LinkedIn 上跨領域的 Remote 工作。
  • 遠端優勢：不受地域限制，甚至可「多兼幾份」。

2) 三個小撇步（@berry_greatful）

  • 搶佔先機（黃金 30 人法則）：
    • 善用關鍵字：Web3 / Remote / Operator / User Growth。
    • 看到新職缺立刻投；申請人數超過 ~30 後回覆率下滑。
  • 英語門檻：
    • JD 用英文屬常態；英語等級可選「Conversational」，夠應付工作溝通即可。
  • 一鍵申請（Quick Apply）：
    • 先優化 Profile、開啟通知；手機滑到適合的就秒投，強化效率。

3) 留言精華（社群補充）

  • 接案/兼職案例：有人做臨床審查（clinical reviewer）按件計酬。
  • 時區與地點：
    • 非 Web3 常要求配合當地時區；美國全遠端 Full-time 常註明需「座標美國」。
    • Web3 職缺較常對跨時區友善。
  • 簽證/合規：部分職缺會要求本地工作簽證或可在該地合法工作。
  • 風險提醒：
    • 防詐騙；留意「無薪實習」或合約不清（簽約前確認薪資、給付、稅務、聘僱型態）。

實用行動清單

  • 搜尋關鍵字：Web3 / Remote / Operator / User Growth。
  • 監控新職缺：開啟職缺通知 + 每日快篩一次。
  • 申請策略：
    • 「<30 申請人」立即投；
    • 先備妥履歷、作品集、Pitch；
    • Quick Apply 節省流程。
  • 簽約前檢查：
    • 地點/時區、薪資（是否實領/幣種/發薪周期）、合約條款、是否需要當地稅務/簽證。
  • 安全與驗證：
    • 檢視公司官網、LinkedIn 員工、Glassdoor 評價；
    • 避免預付費或不合常理要求。
