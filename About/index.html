About

Hi, I’m Harrison 👋

Harrison Chen / Haox — a game developer based in Taipei, Taiwan.

I’ve spent 7+ years at Akatsuki Taiwan building games with Silver Lining Studio — shipping titles like Behind the Frame and The Star Named EOS — and more recently going full-stack: architecting Go backend services and LLM-powered tools as a solo engineer.

I love games, I make games, and when I’m not doing either, I’m probably at a concert.

📮 contactme [at] harrison-chen [dot] dev

What I Do

  • Game Development — Unity (7+ years): gameplay, character animation (Playables / Spine), shaders & rendering, audio, input systems. Shipped on Steam, iOS/Android, Switch, and PS4/PS5.
  • Backend — Go & Python: high-throughput transactional services, FastAPI microservices — owning architecture, security, and deployment end to end as sole engineer.
  • AI / LLM — LangChain multi-agent orchestration; built an internal enterprise knowledge assistant solo, from prototype to release.
  • DevOps — GitHub Actions CI/CD: fully automated Unity builds with Steam deployment, Docker, GCP.
